# **App Name**: StaffFlow

## Core Features:

- Client Booking Requests: Clients can log in and submit requests specifying job role, shift times, location, pay rate, and the number of candidates needed. The tool generates a booking confirmation and job sheet automatically. (Agency booking team also can do it  )
- Smart Roster Management: Recruiters can view confirmed bookings and assign candidates based on availability and compliance. The system updates the roster, sends shift notifications to candidates, and allows clients to view assigned staff. The tool includes drag-and-drop scheduling and optional geolocation check-in.
- Timesheet Management: Track and approve hours worked.
- Automated Invoicing & Payroll: The tool automatically generates invoices for clients and payslips for candidates, using timesheets pulled from the Roster Management System. It calculates invoice amounts, candidate pay, and agency margins, and then automatically generates invoices.
- AI-Powered Candidate Matching: The system suggests the best candidates for bookings using generative AI tool by considering skills, availability, and compliance.
- Interview Scheduling System: Facilitates automated interview scheduling between clients and candidates, integrating calendar availability, video call platforms (Zoom, Teams), and feedback collection.
- Candidate Availability Dashboard: Candidates can update their availability, preferred locations, and roles, which recruiters can view in a live dashboard. The tool uses automated prompts and calendar views.
- Automated Compliance Verification: Candidates upload compliance documents such as RTW, DBS certificates, passports, and visas. A compliance officer verifies the documents, and the tool tracks expiry dates and sends alerts. Only compliant candidates are eligible for bookings, and a compliance dashboard is available for recruiters.
- Candidate Onboarding: Collect personal & job-readiness data
- Client Portal: Manage bookings, invoices, and approvals
- CRM / Relationship Management: Track all interactions and history
- Reporting & Analytics Dashboard: Monitor performance and KPIs

## Style Guidelines:

- Primary color: Sky blue (#87CEEB), a calm and professional color that inspires confidence and symbolizes trust.
- Background color: Light cyan (#E0FFFF), a light, desaturated color to create a clean and uncluttered look.
- Accent color: Lavender (#E6E6FA) complements the sky blue, provides a modern touch, and adds visual interest without distracting from the core functionality.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern and approachable aesthetic suitable for all text elements.
- Simple and professional icons representing each module and function.
- Clean and intuitive layout, prioritizing easy navigation and accessibility.
- Subtle animations to indicate loading or status updates.
